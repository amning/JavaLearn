### Hibernate 多对多操作
#### 多对多配置
1. 创建实体类：用户角色
2. 让两个类之间互相表示
    1. 一个用户里面表示所有角色
    2. 一个角色有多个用户
    * 注：以上均使用set集合进行表示
3. 配置映射关系
    1. 基本配置
        * 不做过多解释
    2. 配置多对多关系
        1. 在用户里面表示所有角色
        2. 在角色里面表示所有用户
        * 均使用set标签
4. 核心文件中引入映射文件

#### 多对多级联保存、删除
###### 根据用户保存角色
1. 用户配置文件中set标签进行配置，cascade值 save-update
2. 代码实现
    * 创建用户和角色对象，把角色放到用户里面，最终保存用户
 ###### 根据用户删除  
 配置cascade = delete  
 级联删除不常用
#### 多对多表格维护(*)
多对多表维护时，只要维护好第三章表就可以了。
在此例子中：
* 让某个用户有某个角色
    * 根据id查询用户和角色
    * 把角色放到用户set集合里面
* 删除某个用户某个角色 
    * 从用户set集合中把角色删掉-->remove方法
